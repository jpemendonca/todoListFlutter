# todo_list

Um aplicativo desenvolvido em Flutter que cria uma lista de tarefas para o dia. Aplicativo desenvolvido seguindo o curso 
"Criação de Apps Android/iOS/Web com Flutter - 5 cursos em 1", do Daniel Ciolfi.
